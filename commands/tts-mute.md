---
description: Temporarily mute TTS notifications
---

Mute TTS notifications for a specified duration. Pass the duration as an argument (e.g., 'for 30 minutes', 'until 3pm', 'for an hour'). To unmute, use 'resume', 'off', 'unmute', or 'cancel'. With no argument, mutes indefinitely.

```bash
${CLAUDE_PLUGIN_ROOT}/scripts/tts-mute.sh $ARGUMENTS
```
